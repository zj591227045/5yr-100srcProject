准备工作

开始配置前，请确保满足以下条件

必需条件
  • Windows 10/11 操作系统
  • 微信PC版已安装并登录
  • 网络连接正常
  • 至少一个AI平台账号
下载安装
  1. 下载 WXAUTO管理工具.exe 程序文件
  2. 双击运行，无需安装
  3. 首次运行会自动创建配置文件
1

添加微信实例

预计用时：2分钟

操作步骤
启动软件
  • 双击 WXAUTO管理工具.exe
  • 等待主界面加载完成
添加实例
  • 点击"实例管理"选项卡
  • 点击左侧"添加本机"按钮
  • 确认配置信息（新手不建议修改）：
实例名称：本机
API地址：http://localhost:5000
轮询间隔：5秒
启用实例：✓ 勾选
启动微信API服务
  • 点击工具栏"启动本地API客户端"按钮
  • 等待 wxauto_http_api.exe 启动
  • 确保微信PC版已登录
验证连接
  • 实例卡片显示绿色表示连接成功（使用wxautox api需要使用激活码激活）
  • 如显示红色，请检查微信API服务是否正常
连接成功标志：实例卡片显示绿色状态
2

配置AI平台

预计用时：2分钟

推荐配置：OpenAI兼容平台
OpenAI兼容平台 推荐

支持OpenaiAPI兼容协议，支持市面上90%的AI服务商

添加平台
  • 选择刚创建的实例
  • 点击中间服务平台区域的"添加平台"按钮
填写配置
平台名称：AI助手（爱写啥写啥）
平台类型：OpenAI
API基础URL：https://api.openai.com/v1（修改为对应服务商的URL）
API密钥：sk-xxxxxxxxxxxxxxxx（服务商提供的API Key）
模型名称：gpt-3.5-turbo（修改为对应服务商的模型）
系统提示词：你是一个有用的AI助手，请简洁友好地回答问题。
测试连接
  • 点击"测试连接"按钮
  • 显示"连接成功"后点击"确定"
其他平台快速配置
Dify平台
平台名称：Dify助手
平台类型：Dify
API Base URL：https://api.dify.ai/v1
API密钥：app-xxxxxxxxxxxxxxxx
关键词匹配
平台名称：自动回复
平台类型：关键词匹配
添加规则：
- 关键词：你好,hi,hello
- 回复：您好！有什么可以帮助您的吗？
3

设置转发规则

预计用时：1分钟

添加规则
  • 在右侧"消息转发规则配置"区域点击"添加规则"按钮
配置规则
规则名称：AI助手规则
微信实例：本机
聊天对象匹配：*（所有聊天对象）/ 测试群（指定群聊）/ 张三（指定好友）多个聊天对象之间使用逗号（英文）分隔
可选：仅处理@消息（勾选后只回复@机器人的消息）
可选：回复时@发送者（勾选后回复时@发送者）
服务平台：AI助手
优先级：10（优先级越高，越先处理）
保存规则
  • 点击"确定"保存配置
规则说明
  • 聊天对象设置：
    • 仅处理指定聊天对象的消息
    • * = 所有聊天对象
    • 测试群 = 指定群聊
    • 张三 = 指定好友
  • @消息处理：
    • 勾选"仅处理@消息"只回复@机器人的消息
    • 不勾选则回复所有消息
4

测试功能

立即验证

发送测试消息
  • 使用任意对象或者指定的聊天对象向托管的微信发送消息
  • 等待AI自动回复
查看处理日志
  • 切换到"消息监听"选项卡
  • 在下方日志区域查看处理过程
验证成功标志
日志显示"消息处理完成"
微信中收到AI回复消息
常见问题快速解决
  • 无回复：检查规则是否启用，平台连接是否正常，修改规则后建议重新启动监听服务
  • 监听服务重启：点击工具栏的结束监听，结束完成后再点击开始监听
  • 回复延迟：正常现象，AI处理需要几秒时间
  • 连接失败：检查API密钥和网络连接
5

启用Web控制台

可选步骤

启动服务
  • 点击工具栏"启动服务"按钮
  • 等待服务启动完成
打开界面
  • 点击"打开界面"按钮
  • 浏览器自动打开Web管理界面
移动端访问
  • 手机浏览器输入：http://电脑IP:8080
  • 例如：http://192.168.1.100:8080
  • 如果电脑没有公网IP，可以使用花生壳等工具将电脑IP映射到公网
